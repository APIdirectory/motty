Motty

Mock your HTTP!

Modern and simpler way to generate custom http responses.

View on github Install motty

Installation.

Python>=3.0 is required, so you should install python 3 before proceeding next step.

Set up an isolated environment. (required)

$ pip3 install virtualenv
$ mkdir ~/mottyenv
$ virtualenv ~/mottyenv
$ source ~/motty/bin/activate

Install motty and play fun!

$ pip install motty
$ run-motty
Hi, motty is now running on http://localhost:7000/
You can stop it by pressing Ctrl or Command + C on the keyboard.
